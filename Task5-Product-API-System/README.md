# Task 5 – Product API System

A full-stack application that manages product data using RESTful APIs. It allows users to add, update, delete, and view products through a backend system integrated with a simple frontend interface.

## Features
- Add new products
- Update existing products
- Delete products
- View product list
- RESTful API endpoints

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## API Endpoints
- GET /products
- POST /products
- PUT /products/:id
- DELETE /products/:id
